# 2048 Fusion

The most beautiful 2048 game ever made. No ads.

**[Play Now](https://carlosgsouza.github.io/2048-fusion/)**

## Install

Works offline. Install it on any device:

- **iOS**: Share > Add to Home Screen
- **Android**: Menu > Install app
- **Desktop**: Click install in your browser's address bar

## Preview

![Neon Theme](public/screenshots/neon.png)

## Controls

- **Arrow keys** / **WASD** — Move
- **Z** / **Ctrl+Z** — Undo
- **Swipe** on mobile

## Development

```bash
git clone https://github.com/carlosgsouza/2048-fusion.git
cd 2048-fusion
npm install
npm run dev
```

Built with Vite, TypeScript, and SCSS.
